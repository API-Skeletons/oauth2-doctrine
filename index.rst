OAuth2 Adapter for Laminas API Tools
====================================

`api-skeletons/oauth2-doctrine <https://github.com/api-skeletons/oauth2-doctrine>`_


This module provides a Doctrine adapter for `laminas-api-tools/mvc-auth <https://github.com/laminas-api-tools/api-tools-mvc-auth>`_
and `laminas-api-tools/oauth2 <https://github.com/laminas-api-tools/api-tools-oauth2>`_
and entity definitions for all aspects of OAuth2 including Authorization Code, Access Tokens, Refresh Tokens, JWT & JTI, and Scopes.

A `Skipper <https://skipper18.com>`_ module is provided to embed in your Entity Relationship Diagram.
